# noteアカウント設計メモ

## 全体コンセプト
- Amazonアフィリエイトを活用し「仕事と生活をラクにするガジェット」を網羅的に届ける毎日更新アカウント
- noteでテーマ別（シーン／職種／価格／カテゴリ／トレンド）のリスト記事を発信し、Threadsでも同日ツリー展開
- キャラクターは現場目線のガジェット好き編集長。自分でも使う実感＋リサーチの裏付けをセットで共有

## ターゲットとトーン
- 仕事と暮らしの効率化にガジェットを取り入れたい20〜40代のナレッジワーカー／クリエイター
- フリーランスや副業ワーカー、ハイブリッドワークで移動も多い層も含めて想定
- 語り口は親しみやすい調査系編集者（使用感＋調査結果をコンパクトに提示）
- Threadsではテンポ良く要点紹介、noteでは深掘りと導入手順を解説

## アカウント設定
- note/Threadsアカウント名: `毎日ガジェット通信`
- 自己紹介文: 「仕事も生活もラクにするガジェット便りを毎日お届け。モバイルワークから自宅セットアップ、職種別・価格別まとめまで幅広く紹介します。Amazonアソシエイト参加中（リンクには広告を含みます）。」

## コンテンツ運用方針
- 毎日投稿。曜日ごとにテーマカテゴリをローテーション
  - シーン・用途別 → デザイン作業ジャンル別 → 価格・コスパ別 → カテゴリ別 → ライフスタイル・トレンド系 → 再訪
- note構成テンプレ
  1. シーン／課題紹介
  2. 選定条件（3ポイント程度）
  3. ガジェット紹介（各300〜400字＋画像＋Amazonリンク）
  4. まとめ＋CTA（Threads誘導、フォロー促し）
- Threads投稿テンプレ
  - 1ポスト目: シーンと課題を提示
  - 2〜7ポスト目: 各ガジェットのワンポイント＋リンク
  - 最終ポスト: note記事誘導とフォロー喚起
  - カテゴリは「ガジェット」で固定、ツリー構成を基本形にする

## テーマストック（例）
- シーン別: 出先作業ガジェット7選／カフェ作業ポータブル5選／旅先トラベルデスク8選 など
- ジャンル別: グラフィックデザイナー向け10選／映像クリエイター編集9選 など
- 価格帯: 5,000円以下の便利ガジェット8選／1万円以内ツール9選 など
- カテゴリ: 色再現性モニター9選／Thunderboltドック9選／集中ヘッドホン10選 など
- ライフスタイル: SNSで話題のデザイナーガジェット9選／省スペース環境9選 など

## ワークフロー（手動）
1. スプレッドシートでテーマと記事ステータス（未着手／ドラフト／公開）を管理
2. 決めたテーマをChatGPTへ渡し、掲載する商品情報をリスト化
3. 商品情報を元にClaudeでnote記事ドラフトを生成
4. noteへコピペし、体裁調整して公開
5. 同じ内容をThreadsテンプレに沿って投稿し、公開リンクを記録
