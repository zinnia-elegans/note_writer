# Threads投稿テンプレート集

> **方針**：切り出し投稿で統一（告知型は使用しない）
> **トーン**：情報メディアとして客観的・中立的に発信

---

## 📌 切り出し投稿テンプレート

### 1. ファクト訴求型

```
{意外な事実・データ}

{カテゴリ}が支持される理由

□ {ポイント1}
□ {ポイント2}
□ {ポイント3}

詳しくはnoteで解説中
```

---

### 2. 比較型

```
{商品A} vs {商品B}

価格：{A価格} vs {B価格}
{比較項目1}：{差異}
{比較項目2}：{差異}

どちらを選ぶべき？noteで解説
```

---

### 3. チェックリスト型

```
{カテゴリ}、どう選ぶ？

□ {選択基準1}で選ぶなら → {商品A}
□ {選択基準2}で選ぶなら → {商品B}
□ {選択基準3}で選ぶなら → {商品C}

詳しくはnoteで比較中
```

---

### 4. 問いかけ型（アンケート）

```
{テーマ}、重視するのは？

A. {選択肢A}
B. {選択肢B}
C. {選択肢C}

コメントで教えてください
```

---

### 5. トレンド型

```
今、{カテゴリ}で注目されているのは

・{商品/ブランド1}
・{商品/ブランド2}
・{商品/ブランド3}

選び方をnoteでまとめました
```

---

### 6. データ型

```
【調査結果】

{カテゴリ}の平均価格：{金額}

{関連データ1}
{関連データ2}

詳しい比較はnoteで公開中
```

---

## ✅ 書き方ルール

| ルール | 内容 |
|-------|------|
| **3行の法則** | 3行以上連続しない |
| **数字を使う** | 価格・個数・期間など |
| **チェックボックス** | □を使って視認性UP |
| **リンク位置** | 必ず最後に配置 |
